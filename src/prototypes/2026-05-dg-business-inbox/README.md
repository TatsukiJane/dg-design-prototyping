# DG Business — главный экран (входящие)

- **Гипотеза:** воспроизведение текущего главного экрана DG Business на нашем prototype-стеке даёт быструю базу для последующих UX-экспериментов (новые статусы, фильтры, групповые действия).
- **Что показываем:** входящие документы в виде таблицы, левый сайдбар с папками, правый рейл с чатами и маркетом, кнопка установки приложения.
- **Открытые вопросы:** какие именно сценарии будем итерировать поверх этого экрана? Массовые действия? Новые фильтры? Информационная плотность строки?
- **Аудитория демо:** продуктовая команда DG Business, для калибровки prototype-стека на «знакомом» интерфейсе.
