# Мессенджер — базовый UI

**Гипотеза:** встроенный мессенджер с поддержкой бота технической поддержки улучшает SX пользователей Documentolog.

**Что показываем:** трёхколоночный layout (список чатов / контекстный сайдбар / диалог), бот с форматированными ответами и кнопками действий.

**Открытые вопросы:** нужен ли контекстный сайдбар или его можно убрать? Какие действия должны быть в хедере чата?

**Аудитория демо:** команда продукта и разработки.
